### 4 Adaptive Layerwise Perturbation: Unifying Off-Policy Corrections for LLM RL

**link**: https://arxiv.org/pdf/2603.19470.pdf
**date**: 2026-03-23
**keywords**: cs.LG
**abs**: 离策略问题（如策略过时和训练-推理不匹配）已成为LLM强化学习（RL）训练稳定性和进一步探索的主要瓶颈。为提高推理效率，推理策略与更新策略之间的分布差距增大，导致重要性权重呈现重尾分布。当策略局部尖锐时会出现重尾权重，这进一步放大尖锐梯度，并可能将更新推到信任区域之外。为此，我们提出自适应分层扰动（ALP），通过在更新期间向每一层的输入隐藏状态注入小的可学习扰动，将其用作目标函数中与未改变的推理策略相对的重要性权重的分子。直观地说，通过向中间表示添加受控噪声，ALP防止更新后的策略与推理策略偏离过大，并扩大策略族以覆盖带有不匹配噪声的推理策略族。因此，扁平化的分布可以自然地缩小更新策略和推理策略之间的差距，并减少重要性权重的尾部，从而保持训练稳定性。实证验证了这一点。在单轮数学任务和多轮工具集成推理任务上的实验表明，ALP不仅提高了最终性能，还避免了迭代训练过程中重要性权重尾部的爆炸和KL峰值，并增强了探索能力。消融实验表明，跨所有层的表示级扰动最为有效，显著优于部分层和仅logits的变体。

### 37 PowerLens: Taming LLM Agents for Safe and Personalized Mobile Power Management

**link**: https://arxiv.org/pdf/2603.19584.pdf
**date**: 2026-03-23
**keywords**: LLM Memory
**abs**: 电池寿命仍是移动设备的关键挑战，然而现有电源管理机制依赖静态规则或粗粒度启发式，忽略了用户活动和个人偏好。本文提出PowerLens系统，利用大型语言模型（LLMs）的推理能力实现安卓设备上安全且个性化的移动电源管理。其核心思想是LLMs的常识推理能够弥合用户活动与系统参数间的语义鸿沟，实现零样本、上下文感知的策略生成，并通过隐式反馈适应个人偏好。该系统采用多代理架构，从UI语义识别用户上下文，生成涵盖18个设备参数的整体电源策略。基于PDL的约束框架在执行前验证每个动作，两级内存系统通过基于置信度的蒸馏从用户隐式覆盖中学习个性化偏好，无需显式配置且3-5天内收敛。实验表明，PowerLens在rooted安卓设备上实现81.7%的动作准确率，比原生安卓节能38.8%，优于基于规则和LLM的基线，具有高用户满意度、快速偏好收敛和强安全保障，系统自身仅消耗每日电池容量的0.5%。
