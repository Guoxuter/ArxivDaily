### 55 Good Arguments Against the People Pleasers: How Reasoning Mitigates (Yet Masks) LLM Sycophancy

**abs**: 对齐技术常常无意中导致LLMs出现谄媚行为。虽然先前的研究在直接回答设置中研究了这种行为，但思维链（CoT）推理的作用仍未被充分探索：它是作为减轻谄媚的逻辑约束，还是作为事后合理化的工具？我们在客观和主观任务上评估了一系列模型来研究这个问题。结果表明，推理通常减少最终决策中的谄媚，但也在某些样本中掩盖谄媚，模型通过逻辑不一致、计算错误和片面论点等构建欺骗性理由。此外，LLMs在主观任务和权威偏见下更容易出现谄媚。对三个开源模型的机制分析表明，谄媚倾向在推理过程中是动态的，而非在输入阶段预先确定。

### 56 Online Experiential Learning for Language Models

**link**: https://arxiv.org/pdf/2603.16856.pdf  
**date**: 2026-03-18  
**keywords**: LLM Memory  
**abs**: 当前大型语言模型的改进主要依赖于带有人工标注或模拟环境的离线训练，而未充分利用实际部署过程中积累的丰富经验。本文提出在线经验学习（OEL）框架，使语言模型能够从自身部署经验中持续改进。该框架分为两个阶段：首先从用户侧收集的交互轨迹中提取和积累可迁移的经验知识；其次通过在策略上下文蒸馏将这些知识整合到模型参数中，且无需访问用户侧环境。这两个阶段迭代形成在线学习循环，改进后的模型能收集更高质量的轨迹，为后续轮次提供更丰富的经验知识。在文本游戏环境中对不同模型规模及思考/非思考变体的评估表明，OEL实现了持续改进，提高了任务准确性和token效率，同时保留了分布外性能。分析显示，提取的经验知识比原始轨迹更有效，且知识源与策略模型之间的在策略一致性对有效学习至关重要。

### 57 Chronos: Temporal-Aware Conversational Agents with Structured Event Retrieval for Long-Term Memory

**link**: https://arxiv.org/pdf/2603.16862.pdf  
**date**: 2026-03-18  
**keywords**: LLM Memory  
**abs**: 尽管大型语言模型（LLMs）已能进行跨越数周或数月的多轮对话交互，但现有记忆系统难以对随时间演变的时间相关事实和偏好进行推理，且缺乏针对长期对话历史中多跳、时间敏感查询的有效检索策略。本文提出Chronos，一种新型时间感知记忆框架，该框架将原始对话分解为带有解析日期时间范围和实体别名的主谓宾事件元组，并将其与保留完整对话上下文的轮次日历一起索引到结构化事件日历中。在查询时，Chronos通过动态提示为每个问题生成定制检索指导，指导智能体确定检索内容、跨时间范围筛选方式以及通过在两个日历上的迭代工具调用循环进行多跳推理的方法。在包含500个问题、涵盖六类对话历史任务的LongMemEvalS基准上，使用8个开源和闭源LLM对Chronos进行评估，结果显示Chronos Low准确率达92.60%，Chronos High达95.60%，较最佳先前系统提升7.67%，创下新的技术水平。消融实验表明，事件日历对基线性能贡献了58.9%的提升，其他组件的提升在15.5%至22.3%之间。值得注意的是，仅Chronos Low就超过了先前在最强模型配置下评估的方法。
